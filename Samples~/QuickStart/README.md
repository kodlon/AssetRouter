# Quick Start Sample

This sample is a placeholder for a future tutorial scene demonstrating Asset Router in action.

## What will be here

- A `SampleScene.unity` with a few `T_*.png`, `UI_*.png`, `SFX_*.wav` test files placed at `Assets/` root.
- A pre-configured `ImporterSettingsDatabase.asset` with the four default rules.
- Step-by-step notes showing how the postprocessor moves files into `Assets/Art/...` and applies the bundled `.preset` files.

## How to import

In Unity, open **Window → Package Manager**, select **Asset Router**, expand the **Samples** section, and click **Import** next to *Quick Start*.

Unity will copy this sample into your project's `Assets/Samples/Asset Router/<version>/Quick Start/`.
